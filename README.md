# MovieFinder

TMDB API를 활용한 영화 검색 웹앱  
Vue 3 기반으로 제작되었으며, 검색/상세조회/즐겨찾기 기능을 제공

##사용 기술
- **Vue 3 + TypeScript + Composition API**
- **Pinia** – 상태 관리
- **Tailwind CSS** – 빠르고 유연한 스타일링
- **Axios** – API 통신
- **TMDB API** – 영화 정보 제공
- **Vite** – 빠른 개발 환경
- **LocalStorage** - 즐겨찾기 저장

##실행 방법
- npm install 
- npm run dev
